# 🌡️ DHT11 Weather Monitor using ESP8266

## 📌 Project Overview

This project is a real-time weather monitoring system built using the ESP8266 NodeMCU and DHT11 sensor. The ESP8266 hosts a local web server that displays live temperature and humidity readings through a user-friendly web dashboard.

## 🚀 Features

- Real-time Temperature Monitoring
- Real-time Humidity Monitoring
- ESP8266 Local Web Server
- Auto Refresh Every 5 Seconds
- Responsive Web Interface
- Wireless Data Monitoring

## 🛠️ Hardware Components

- ESP8266 NodeMCU
- DHT11 Temperature & Humidity Sensor
- Jumper Wires
- USB Cable

## 🔌 Circuit Connections

| DHT11 Pin | ESP8266 Pin |
|-----------|------------|
| VCC | 3.3V |
| GND | GND |
| DATA | D4 (GPIO2) |

## 💻 Software Used

- Arduino IDE
- ESP8266 Board Package
- DHT Sensor Library
- HTML & CSS
